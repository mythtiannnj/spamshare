How to Get Facebook Cookie

Follow these steps to extract your Facebook cookie array for automated sharing.

Step 1 Download Extension

Download the Biar Utility extension from the link below:

Download Extension (crx file)
Click to download the CRX file

Step 2 Login & Open Menu

Login to your dummy Facebook account on Kiwi Browser. Click the 3 dots in the upper right corner.

Click the three dots menu

Step 3 Access Extensions

Click on "Extensions" from the dropdown menu.

Select Extensions from the menu

Step 4 Load Extension

Click on "+ from .zip/.crx/user.js" and select the downloaded CRX file.

Load the extension file

Step 5 Select the File

Browse and select the extension file you downloaded.

Select the CRX file

Step 6 Activate Extension

Click the "Activate" button to enable the extension.

Click Activate to enable

Step 7 Get Cookie String

Go back to Facebook, click the 3 dots again, and scroll to the bottom option. Click it to copy the cookie string.

Copy the cookie string

Step 8 Copy & Use

Your cookie array is now copied! Paste it into the dashboard form.

Cookie successfully copied
Example format: [{"key":"sb","value":"xxx"},{"key":"datr","value":"xxx"}]

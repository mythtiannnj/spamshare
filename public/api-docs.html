Simplified Share

GET /share
Start a sharing session with a single GET request. Cookies must be passed in the Cookie or x-cookie header.
link *required — Facebook post URL amount *required — Number of shares (1-10000) delay — Delay between shares (seconds, default: 2)
GET /api/share?link=https://www.facebook.com/...&amount=10&delay=2
Header: Cookie: sb=xxx; datr=xxx; ... 

                            

                        

                    

                

            


            
            

                
 Simplified Share (All-in-One)

                

                    

                        GET
                        /ap/share
                    

                    
Start a sharing session with a single GET request. All parameters including cookie are passed in the query string.

                    

                        cookie *required — Facebook cookie string (URL-encoded)
                        link *required — Facebook post URL
                        amount *required — Number of shares (1-10000)
                        delay — Delay between shares (seconds, default: 2)
                    

                    

                        
                        GET /ap/share?cookie=sb%3Dxxx%3B%20datr%3Dxxx&link=https%3A%2F%2Fwww.facebook.com%2F...&amount=10&delay=2
                    


                    
                    

                        
                        

                            

                                

                                    
                                    
                                

                            

                            

                                

                                    
                                    
                                

                                

                                    
                                    
                                

                                

                                    
                                    
                                

                            

                            
                            

                                
                                
                            

                            

                                

                                    
                                    
                                

                                

                            

                        

                    

                

            


            
            

                
 Other Endpoints


                

                    
POST /submit

                    
Start a sharing session (legacy POST method).

                    

                        cookie *required — JSON or cookie string
                        url *required — Post URL
                        amount *required — Shares (1-10000)
                    

                    

                         POST /api/submit { "cookie": "...", "url": "...", "amount": 10 }
                    

                


                

                    
GET /sessions/summary

                    
Get summary of all active and completed sessions.

                    

                         GET /api/sessions/summary
                    

                


                

                    
GET /session/:sessionId/logs

                    
Retrieve logs for a specific session.

                    

                         GET /api/session/abc123/logs
                    

                


                

                    
DELETE /session/:sessionId

                    
Stop a running session.

                    

                         DELETE /api/session/abc123
                    

                


                

                    
GET /tiktok/random

                    
Get a random TikTok video URL.

                    

                         GET /api/tiktok/random
                    

                


                

                    
GET /health

                    
Check server health and active sessions.

                    

                         GET /api/health
                    

                

            


            
            

                
 Authentication

                

                    All endpoints require a valid Facebook cookie for authentication. You can obtain your cookie using the 
                    tutorial.
                    For the GET /share endpoint, pass the cookie in the Cookie header or x-cookie header.
                    For POST endpoints, include the cookie in the request body.
